# החנות שלנו — אתר קטלוג מוצרים

אתר שמרכז את כל המוצרים שבמלאי בצורה מסודרת ונוחה ללקוחות, כולל מחירים, עגלת קניות ומערכת קופונים.

## תכונות

- **קטלוג מוצרים** מסודר לפי קטגוריות (אביזרים, בלוני מים) עם מחיר לכל פריט.
- **חיפוש** מהיר של מוצרים לפי שם.
- **עגלת קניות** — הוספה/הפחתה/הסרה של מוצרים, חישוב סכום ביניים אוטומטי.
- **מערכת קופונים** — קוד קופון `SAVE15` מעניק הנחה של **15 ₪** על הרכישה.
- ממשק בעברית (RTL), מותאם למובייל.

ניתן להוסיף קופונים נוספים בקובץ `src/products.js` (אובייקט `coupons`).

## הרצה מקומית

```bash
cd catalog
npm install
npm run dev
```

האתר ייפתח בכתובת שתודפס בטרמינל (בדרך כלל http://localhost:5173).

## בנייה לפרודקשן

```bash
npm run build      # יוצר תיקיית dist/
npm run preview    # תצוגה מקדימה של הבילד
```
